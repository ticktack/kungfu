一个专为JFinal设计的构建RESTful Web服务的急速开发库。
## 特性
1. 能够快速构建 JFinal RESTful API 项目。
2. 支持代码生成能力，使用极度简单及灵活，常规的代码帮您写。
3. 支持动态构建查询SQL语句。
4. 加速Controller层开发的速度及体验。
5. 加速Service层开发的速度及体验。
6. 更多特性等你发现...
